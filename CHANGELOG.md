# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-13

### Added
- Enterprise starter architecture for a self-healing cloud platform.
- FastAPI orchestrator with anomaly detection, policy decisions, and remediation simulation.
- Streamlit dashboard for operations visibility and test signal injection.
- Prometheus scrape configuration and Docker Compose baseline.
- Windows PowerShell scripts for install/start/stop and LAN URL access.
- GitHub project essentials: MIT license, CI workflow, contribution guide, issue templates, and pull request template.
